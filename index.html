02

What I actually do

Problem → action → result. Not a pitch deck — three snapshots of operator work with numbers attached.

Commerce growth

TikTok CPI $0.09, paid CAC $8

Problem: Consumer commerce funnel with thin margins — standard CAC math was lying to us.

Action: Rebuilt acquisition math across Meta and TikTok, 80 creatives/week pipeline, funnel analysis by cohort.

Result: $0.09 TikTok CPI and $8 paid CAC per buyer in live commerce experiments — with documented unit economics, not vanity metrics.

AI in production

Stable Diffusion API in 2 days

Problem: Competitor shipped an AI image product; we had no backend and two weeks felt too long.

Action: Serverless pipeline on AWS — Lambda, API Gateway, model hosting, cost-aware scaling from day one.

Result: Production API live in 48 hours. The hard part wasn't the build — it was everything after launch.

Infrastructure

Serverless at founder scale

Problem: Tiny teams can't carry heavy infra — but still need production-grade reliability across 10+ products.

Action: Repeatable serverless architecture: Lambda, DynamoDB, S3, CloudFront, IaC, CI/CD — tuned for cost and iteration speed.

Result: Same stack from Hypee through cheap.market. AWS Solutions Architect certified; I still write the production code myself.
